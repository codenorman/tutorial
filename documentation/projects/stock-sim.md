# stock-sim 

## Create project

```shell script
ng new stock-sim --routing --style css

```

## Install dependencies

```shell script
ng add @briebug/jest-schematic
npm i -D jasmine @types/jasmine prettier

```
## install storybook

```shell script
npx -p @storybook/cli sb init
```

## start application

```shell script
ng serve stock-sim
```
